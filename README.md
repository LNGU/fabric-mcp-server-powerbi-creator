# Fabric MCP Server
An Model Context Protocol (MCP) server for managing Microsoft Fabric workspaces, deployments, and report generation.

## Features
- **List Workspaces:** View available Fabric workspaces.
- **Git Sync:** Trigger `updateFromGit` to deploy changes.
- **Report Generation:** Create reports from templates.
- **Theme Generator:** Apply custom branding (colors/fonts) to reports.

## Installation
```bash
pip install fabric-mcp-server
```

## Usage (MCP Client)
Connect your AI agent to this server to manage Fabric resources.
